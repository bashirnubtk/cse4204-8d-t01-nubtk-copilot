## 📌 Project Overview
**NUBTK_COPILOT** is a next-generation Academic Resource Planning (ERP) platform and automated career architecture framework. While standard university systems only track aggregate attendance and grades, NUBTK_COPILOT granularly logs the technical skill inventories acquired by students across semesters. By analyzing student progress against real-time recruitment metrics, it automatically bridges the structural gap between classical academia and modern industry hiring expectations.

## 🛠️ Core Architectural Pillars (Features)
* **Contextual AI Mentor Hub (OpenRouter / Claude 3):** Seamless continuous asynchronous mentoring engine delivering technical program troubleshooting, system code optimization advice, and architectural learning roadmaps.
* **Predictive Admin Insight Engine (Scikit-Learn):** Machine learning statistical diagnostic modules tracking continuous assessment indexes to flag academically vulnerable students before formal term evaluations.
* **Algorithmic Career Readiness Scoring:** Dynamic live evaluation engine producing weighted index maps scaling continuous academic metrics against active employment job boards (e.g., BDJobs) APIs.

## ⚙️ Development Environment Setups

Follow these steps to run the core server registry locally:

1. Clone the master repository system:

```bash
git clone [https://github.com/your-username/cse4204-8d-t03-nubtk-copilot.git](https://github.com/your-username/cse4204-8d-t03-nubtk-copilot.git)
cd cse4204-8d-t03-nubtk-copilot

```

2. Initialize isolated virtual execution stack:

```bash
python -m venv venv
# Activate on Windows systems:
.\venv\Scripts\activate

```

3. Fetch core framework dependencies:

```bash
pip install -r requirements.txt

```

4. Complete migrations and run local platform server:

```bash
python manage.py migrate
python manage.py runserver

```
